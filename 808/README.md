# 808

Descripción:

Grosso modo este proyecto se encarga de generar codigos de barra en formato 128 por medio de la libreria ZXing en base a 
los artículos de la combinación de familia-linea elegida en la interfaz para posteriormente cargar los codigos de barras 
en un PDF para finalmente cargar el pdf en un visor.

Objetos SQL:

SpVTASGeneradorCodigoBarras

Notas:

Este proyecto asincrono se hizo en .Net Framework - winforms, usa los controles estilizados del framework Guna
,interactua con la BD por medio de dapper y se empleo la arquitectura MVVM.

En caso de tener algún error de DLLs, usar las DDL en la carpeta "Resources/DLL/".
